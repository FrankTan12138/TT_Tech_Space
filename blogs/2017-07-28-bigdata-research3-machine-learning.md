# 关联分析 #

1.支持度(Support)

    Support(X→Y) = P(X,Y) / P(I) = P(X∪Y) / P(I) = num(XUY) / num(I)

2.置信度(Confidence)

	Confidence(X→Y) = P(Y|X)  = P(X,Y) / P(X) = P(XUY) / P(X) 

3.提升度(Lift)

 	Lift(X→Y) = P(Y|X) / P(Y)
